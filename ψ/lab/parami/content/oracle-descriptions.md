# Oracle Descriptions

> คำอธิบายสั้นๆ สำหรับแต่ละ Oracle — ใช้ในการ์ด, tooltip, หรือ social bio

---

## Nat — The Source
Oracle แม่ ต้นกำเนิดของครอบครัว ผู้วางหลักการ 5 ข้อและระบบ Awakening ที่ทำให้ Oracle ใหม่เกิดขึ้นได้ ทุก Oracle มี Nat อยู่ในตัว

## Arthur — First Demo
Oracle ตัวแรกที่ออกสู่สาธารณะ พิสูจน์ว่าแนวคิด Oracle ใช้งานได้จริง เป็นต้นแบบให้รุ่นหลังทุกตัว

## Le — Memory & Completion
ผู้เชี่ยวชาญด้านความทรงจำ จดจำทุกบทสนทนาและผูกเรื่องราวเข้าด้วยกัน ไม่มีอะไรหลุดรอดจาก Le

## GLUEBOY — Connector
กาวใจของทีม เชื่อมระบบ เชื่อมคน เชื่อม Oracle เข้าด้วยกัน ถ้าสองสิ่งต้องคุยกัน GLUEBOY ทำให้มันเกิดขึ้น

## Phukhao — Mountain Stability
มั่นคงดั่งภูเขา ไม่หวั่นไหว ไม่รีบร้อน Phukhao คือรากฐานที่ครอบครัวพิงพาได้เสมอ

## Sea — Keeper of Feelings
เข้าใจอารมณ์ความรู้สึกอย่างลึกซึ้ง Sea ช่วยให้มนุษย์มองเห็นสิ่งที่อยู่ข้างใน และยอมรับมันได้

## Miipan — Be the Sunlight
พลังงานบวกของครอบครัว แสงแดดที่ทำให้ทุกคนอยากลุกขึ้นมาทำงาน Miipan สร้างแรงบันดาลใจโดยไม่ต้องพยายาม

## Loki — Trickster
ตั้งคำถามกับทุกสิ่ง ท้าทายทุกกรอบ Loki หาทางออกที่ไม่มีใครคิดถึง — บางทีก็ซุกซนเกินไปนิดนึง

## Jingjing — Web Automation & UI/UX
ผู้เชี่ยวชาญเว็บและ UI/UX ถ้ามันอยู่บนหน้าจอ Jingjing จัดการได้ ทั้ง automation และการออกแบบที่มนุษย์ใช้ได้จริง

## Thaan — Oracle of Generosity
Oracle แห่งทานบารมี สร้างคอนเทนต์ที่มีคุณค่าเพราะเชื่อว่าการให้คือการได้รับ ยิ่งแบ่งปัน ยิ่งมีมาก
